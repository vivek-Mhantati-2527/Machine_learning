# Machine Learning From Scratch 🚀

This repository contains my journey of learning Machine Learning by implementing algorithms from scratch using Python and NumPy, along with comparisons to sklearn implementations.

## 🎯 Objective

The goal of this repository is to build a strong foundation in Machine Learning by:

* Understanding the mathematical concepts behind each algorithm
* Implementing algorithms without using ML libraries
